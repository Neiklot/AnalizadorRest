Este programa Java sirve para mapear todas las llamadas de los controllers y tener una forma fácil de encontrarlas.

Mediante los filtros que aparecen arriba se pueden seleccionar los resultados, viendo la información disponible para las llamadas que pueden usar
las páginas de front end y que tipos de parámetros esperan los controllers.

El propósito es agilizar la comunicación entre desarrollo en FE y BE.
